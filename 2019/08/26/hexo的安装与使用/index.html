hexo的安装与使用

首先安装node.js
切换镜像源npm install -g cnpm –registry=https://registry.npm.taobao.org
安装hexo cnpm install -g hexo-cli
hexo -v 查看hexo的版本
切换到/c/Users/用户名称 创建目录为myblog
cd myblog
初始化博客 hexo init
启动博客 hexo s (一般在本地做预览用)
创建博客 hexo n “我的第一篇博客”
清理一下 hexo clean
生成文章 hexo g
启动预览 hexo s
$ hexo s
INFO Start processing
INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

这样hexo就能在本地的浏览器中查看了
